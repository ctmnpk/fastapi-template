# Apresentation
This repository purpose is to present a simple API using [FastAPI](https://fastapi.tiangolo.com/#example) webframework. Here i only handle with the framework on subject to serve as a backend for a TO-DO application, hence it does basic authentication, uses `GET`, `POST`, `PUT` and `DELETE` restful methods, data modelling with [PyDantic](https://docs.pydantic.dev/latest/) and approachs [docker](https://docs.docker.com/) usage.
